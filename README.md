# Desmos-Custom-Function-Extension

A small browser extension that lets you integrate your own JavaScript functions into Desmos via a user interface. 

---

## How to Install

Because this extension is hosted on GitHub, you install it as an "unpacked" extension. Follow these quick steps:

### Step 1: Download the Extension
1. Click the green **Code** button at the top right of this page.
2. Select **Download ZIP**.
3. Extract (unzip) the downloaded file to a folder on your computer.

### Step 2: Enable Developer Mode in Chrome
1. Open Google Chrome and navigate to `chrome://extensions/`.
2. In the top-right corner, switch the **Developer mode** toggle to **ON**.

### Step 3: Load the Extension
1. Click the **Load unpacked** button in the top-left corner.
2. Select the extracted folder (make sure to select the root folder containing the `manifest.json` file).
